
 SUMMER LANGUAGE(The_base):

   make -f Makefile_JIT clean
   make -f Makefile_JIT

 To compile with( VM ) type:
   make -f Makefile_VM clean
   make -f Makefile_VM

 To test the Summer Language(WINDOWS), type:
   summer.exe test.s

 To test the Summer Language(LINUX), type:
   ./summer test.s
