 SUMMER LANGUAGE:

 ----------------------------------------------

 01: To compile SUMMER type:
   make clean
   make

 02: To compile SUMMER WITH APPLICATION API type:
   make -f Makefile_app clean
   make -f Makefile_app

 To test the SUMMER program, type:
   sum test.sum

 To test the SUMMER WITH APPLICATION API, type:
   app app.app
   or
   app
